# صفحة Homepage - قالب مبدئي

قمت بإنشاء قالب HTML/CSS/JS بسيط للصفحة الرئيسية (Hero + Navbar + 3 Cards + News ticker).

كيفية التشغيل محليًا:


اقتراحات التطوير التالية:

قمت بإنشاء قالب HTML/CSS/JS محسّن للصفحة الرئيسية (Hero + Navbar + 3 بطاقات + News ticker)، وأضافت زخرفة SVG وخريطة SVG ومودال فيديو تجريبي.

كيفية التشغيل محليًا:

- افتح الملف `index.html` في المتصفح (نقرة مزدوجة أو فتح عبر المتصفح).

الأصول الجديدة المضافة إلى مجلد `assets/`:

- `pattern.svg` — نقش زخرفي متكرر للخلفية
- `map.svg` — خريطة توضيحية للاستخدام في الـ Hero (استبدال لاحقًا بخريطة مفصّلة)
- `favicon.svg` — أيقونة الموقع

اقتراحات التطوير التالية:

- استبدال الخريطة والأنماط بأصول عالية الدقة (SVG أو WebP).
- إضافة مشغّل فيديو داخل المودال (`#introModal`) مع lazy-loading و poster.
- نقل المشروع إلى React/Vite أو Next.js إذا رغبت بخدمات SSR وCMS.
- إعداد نظام تسليم الأصول (Figma handoff) وتجهيز ملفات SVG منفصلة للأيقونات.
